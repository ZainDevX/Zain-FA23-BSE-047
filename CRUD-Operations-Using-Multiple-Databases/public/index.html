CRUD with Multiple Databases

A single Node.js application performing Create, Read, Update & Delete operations across MongoDB, MySQL, and SQLite databases.

M

MongoDB

NoSQL document database. Uses Mongoose ODM for schema validation and queries.
S

MySQL

Relational database with structured tables. Uses mysql2 driver with prepared statements.
L

SQLite

Lightweight file-based database. Uses better-sqlite3 for synchronous, fast operations.
